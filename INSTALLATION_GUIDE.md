# Installation Guide

## Prerequisites

- Flutter SDK installed and available on your PATH
- Dart SDK bundled with Flutter
- Android Studio or VS Code with Flutter extensions
- Android emulator, iOS simulator, or a connected device

## Setup

1. Install dependencies:

```sh
flutter pub get
```

2. Verify the environment:

```sh
flutter doctor
```

3. Run the app:

```sh
flutter run
```

## Quality Checks

Run static analysis:

```sh
flutter analyze
```

Run the test suite:

```sh
flutter test
```
